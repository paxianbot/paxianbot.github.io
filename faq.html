FAQ & How to Use

  • What is Paxian?
    • Paxian is a versatile Discord bot designed to enhance your server experience with a variety of commands and features.
  • How do I invite Paxian to my server?
    • You can invite Paxian to your server using the following link: Invite Paxian.
  • How do I report a bug or suggest a feature?
    • You can report bugs or suggest features by joining our support server: Support Server.
  • Is my data safe with Paxian?
    • Yes, we only collect public data as (server id and user id) and Paxian takes privacy and data security seriously. Please refer to our Privacy Policy for more details. .
  • Registering Your Guild and User
    • Register Guild: Use the /register-guild command to register your guild with the bot. (Administrator only)
    • Register User: Use the /register-user command to register yourself with the bot.
  • Managing Item Lists
    • Add Item List: Use the /add-item-list command to add items to your list.
    • Delete Item from List: Use the /delete-item-from-list command to remove an item from your list.
    • List Item Lists: Use the /list-item-lists command to view all your item lists.
  • Managing Professions
    • Add Profession: Use the /add-profession command to add a new profession.
    • Update Profession: Use the /update-profession command to update an existing profession.
    • Delete Profession: Use the /delete-profession command to delete a profession.
    • List Professions: Use the /list-profession command to view all professions.
  • Localization
    • Manage Localization: Use the /manage-localization command to manage localization settings. (Administrator only)
  • Additional Commands
    • Help: Use the /help command to get a list of all available commands and their descriptions.
    • List by Profession: Use the /list-by-profession command to list items by profession.
  • Unregistering
    • Unregister Guild: Use the /unregister-guild command to unregister your guild from the bot.
    • Unregister User: Use the /unregister-user command to unregister yourself from the bot.
